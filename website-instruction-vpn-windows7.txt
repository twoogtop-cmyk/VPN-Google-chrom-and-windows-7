<section>
  <h2>Важно перед установкой</h2>
  <ul>
    <li>Поддерживается: Windows 7, Windows 10, Windows 11.</li>
    <li>Нужен Google Chrome.</li>
    <li>
      Расширение может <strong>не добавиться автоматически</strong> в обычный Chrome из-за политики безопасности браузера.
      Это нормально. Ниже есть отдельный шаг «Как добавить расширение вручную за 1 минуту».
    </li>
  </ul>
</section>

<section>
  <h2>Шаг 1. Скачайте и распакуйте архив</h2>
  <ol>
    <li>Скачайте архив по ссылке выше.</li>
    <li>Нажмите правой кнопкой по архиву → «Извлечь все».</li>
    <li>Откройте распакованную папку.</li>
  </ol>
</section>

<section>
  <h2>Шаг 2. Запустите установщик</h2>
  <ol>
    <li>Запустите файл <strong>install-windows.bat</strong> (лучше «Запуск от имени администратора»).</li>
    <li>Подождите, пока установка полностью завершится.</li>
    <li>Если появится окно контроллера с текстом <code>Controller listening on http://127.0.0.1:23999</code>, это хорошо — сервис работает.</li>
  </ol>
</section>

<section>
  <h2>Шаг 3. Если расширение не появилось автоматически в Chrome</h2>
  <p><strong>Это самый частый вопрос.</strong> Сделайте вручную:</p>
  <ol>
    <li>Откройте в Chrome страницу <code>chrome://extensions</code></li>
    <li>Включите переключатель <strong>Режим разработчика</strong> (справа сверху).</li>
    <li>Нажмите <strong>Загрузить распакованное расширение</strong>.</li>
    <li>Выберите папку:<br /><code>C:\Users\%USERNAME%\AppData\Local\VlessChromeVpn\extension</code></li>
    <li>Появится расширение <strong>VLESS VPN</strong>.</li>
  </ol>
  <p>
    После этого расширение добавлено и готово к работе.
  </p>
</section>

<section>
  <h2>Шаг 4. Закрепите иконку расширения (чтобы не искать каждый раз)</h2>
  <ol>
    <li>Нажмите значок «пазл» в правом верхнем углу Chrome.</li>
    <li>Найдите <strong>VLESS VPN</strong>.</li>
    <li>Нажмите значок «булавка» — иконка закрепится рядом с адресной строкой.</li>
  </ol>
</section>

<section>
  <h2>Шаг 5. Получите VLESS ссылку</h2>
  <ol>
    <li>Откройте личный кабинет: <a href="https://cp.sevenskull.ru/login" target="_blank" rel="noopener noreferrer">https://cp.sevenskull.ru/login</a></li>
    <li>Войдите в аккаунт.</li>
    <li>Создайте тикет в поддержку с запросом на выдачу VLESS конфигурации.</li>
    <li>После ответа поддержки скопируйте вашу ссылку формата <code>vless://...</code>.</li>
  </ol>
</section>

<section>
  <h2>Шаг 6. Подключите VPN</h2>
  <ol>
    <li>Нажмите на иконку <strong>VLESS VPN</strong> в Chrome.</li>
    <li>Вставьте ссылку в поле <strong>VLESS ссылка</strong>.</li>
    <li>Нажмите <strong>Сохранить</strong>.</li>
    <li>Нажмите <strong>Включить</strong>.</li>
  </ol>
  <p>Если всё успешно — увидите статус «Включено».</p>
</section>

<section>
  <h2>Шаг 7. Проверьте, что всё работает</h2>
  <ol>
    <li>В расширении нажмите <strong>Проверить контроллер</strong>.</li>
    <li>Потом нажмите <strong>Проверить подключение</strong>.</li>
    <li>Если тест показывает текущий IP — VPN работает.</li>
  </ol>
</section>

<section>
  <h2>Как отключить VPN</h2>
  <p>Откройте расширение и нажмите <strong>Выключить</strong>.</p>
</section>

<section>
  <h2>Очень частые вопросы и ответы</h2>

  <h3>1) Почему расширение не установилось само?</h3>
  <p>
    В обычном Google Chrome автоматическое «тихое» добавление расширений ограничено политикой безопасности.
    Поэтому иногда нужно один раз добавить расширение вручную через <code>chrome://extensions</code>.
  </p>

  <h3>2) Кнопка «Включить» нажимается, но интернета нет</h3>
  <ul>
    <li>Проверьте, что ссылка начинается с <code>vless://</code> и вставлена полностью.</li>
    <li>Запустите ярлык <strong>VLESS VPN Restart Controller</strong>.</li>
    <li>В расширении снова нажмите «Проверить контроллер».</li>
    <li>Проверьте дату и время Windows (особенно на Windows 7).</li>
  </ul>

  <h3>3) Ошибка «Контроллер недоступен»</h3>
  <ul>
    <li>Запустите ярлык <strong>VLESS VPN Restart Controller</strong>.</li>
    <li>Если не помогло — перезапустите компьютер.</li>
    <li>Откройте расширение и снова выполните проверку.</li>
  </ul>

  <h3>4) Антивирус ругается на установщик</h3>
  <p>
    Иногда защитник Windows блокирует запуск скриптов. Добавьте папку с установщиком в исключения или разрешите запуск вручную.
  </p>

  <h3>5) Работает ли это как VPN для всей системы Windows?</h3>
  <p>
    Нет. Это VPN/прокси для <strong>Google Chrome</strong>. Другие программы Windows через него не идут.
  </p>

  <h3>6) Где хранится VLESS ссылка?</h3>
  <p>Локально в браузере (в хранилище расширения).</p>

  <h3>7) Где получить VLESS ссылку?</h3>

  </p>
