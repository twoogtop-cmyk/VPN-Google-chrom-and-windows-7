# Политика конфиденциальности VLESS VPN Switch

Дата вступления в силу: 23.02.2026

## Какие данные обрабатывает расширение

Расширение VLESS VPN Switch обрабатывает только данные, необходимые для работы VPN-функций в браузере Chrome:

- введенная пользователем VLESS-ссылка;
- техническое состояние подключения (включено/выключено, номер локального SOCKS-порта, текст последней ошибки).

Эти данные хранятся локально в `chrome.storage.local` на устройстве пользователя.

## Какие данные НЕ собираются

Расширение не собирает и не передает разработчику:

- историю посещений;
- содержимое страниц;
- пароли, сообщения, платежные данные;
- персональные данные пользователя.

## Передача данных третьим сторонам

Расширение не отправляет пользовательские данные на сервер разработчика.

Для работы используются:

- локальный контроллер на `127.0.0.1`;
- внешний VPN-сервер, указанный пользователем в VLESS-конфигурации;
- проверка IP при тесте подключения через `https://api.ipify.org/?format=json` (только по запросу пользователя из popup).

## Хранение и удаление данных

Пользователь может удалить сохраненные данные:

- отключив/удалив расширение;
- очистив данные расширения в Chrome.

## Изменения политики

Политика может обновляться при изменении функциональности расширения.
