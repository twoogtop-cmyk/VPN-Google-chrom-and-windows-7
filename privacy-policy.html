Политика конфиденциальности VLESS VPN Switch

Дата вступления в силу: 23.02.2026

1. Какие данные обрабатывает расширение

Расширение обрабатывает только данные, необходимые для работы VPN-функций в Chrome:

  • введенная пользователем VLESS-ссылка;
  • техническое состояние подключения (включено/выключено, локальный SOCKS-порт, последняя ошибка).

Данные сохраняются локально в chrome.storage.local на устройстве пользователя.

2. Какие данные НЕ собираются

  • история посещений;
  • содержимое страниц;
  • пароли, сообщения, платежные данные;
  • персональные данные пользователя.

3. Передача данных третьим сторонам

Расширение не отправляет пользовательские данные на сервер разработчика.

Для работы используются:

  • локальный контроллер на 127.0.0.1;
  • внешний VPN-сервер из VLESS-конфигурации пользователя;
  • проверка IP через https://api.ipify.org/?format=json только по кнопке теста в popup.

4. Хранение и удаление данных

Пользователь может удалить данные, отключив/удалив расширение или очистив данные расширения в Chrome.

5. Изменения политики

Политика может обновляться при изменении функциональности расширения.
