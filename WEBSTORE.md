# Публикация в Chrome Web Store

## Важно заранее

Это расширение использует локальный контроллер и Xray (устанавливаются в Windows отдельно).  
То есть из магазина расширение может быть установлено в 1 клик, но для работы VPN пользователю все равно нужен локальный установщик.

## Что подготовить

1. Иконки расширения (минимум 128x128 для магазина)
2. Скриншоты popup
3. Политику конфиденциальности (URL)
4. ZIP с папкой `extension`

Готовые тексты политики находятся в:

- `PRIVACY_POLICY.md`
- `privacy-policy.html`

Рекомендуется разместить `privacy-policy.html` на вашем домене и вставить этот URL в Chrome Web Store.

## Как собрать ZIP

Содержимое ZIP должно быть файлами расширения, а не оберткой из лишней папки.

Готовый скрипт сборки:

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\package-webstore.ps1
```

ZIP будет создан в `dist/vless-vpn-switch-extension.zip`.

## Черновик описания для магазина

Название: **VLESS VPN Switch**

Кратко:
Расширение для быстрого включения VLESS VPN в Chrome. Вставьте VLESS ссылку и нажмите кнопку включения.

Полное описание:
- Вставка VLESS ссылки из личного кабинета
- Включение/выключение VPN в один клик
- Автовосстановление сессии после перезапуска Chrome
- Проверка контроллера и тест подключения в popup
- Получить ключ бесплатно на 3 дня. Ежемесячная подписка от 169 рублей
- Личный кабинет: https://cp.sevenskull.ru/login

Важное примечание для пользователя:
Для работы требуется локальный компонент (контроллер + Xray).  
Скачайте и запустите companion-установщик для Windows: `install-marketplace-windows.bat`.

## Права и обоснование

- `proxy`: включение прокси Chrome для маршрутизации трафика через локальный SOCKS
- `storage`: сохранение VLESS ссылки и состояния
- `alarms`: периодическая проверка состояния VPN
- `host_permissions` `127.0.0.1`: связь с локальным контроллером

## Рекомендации по модерации

- Укажите, что расширение не собирает персональные данные
- Укажите, что VLESS ссылка хранится локально в браузере
- Четко опишите, что это не системный VPN, а прокси именно для Chrome
